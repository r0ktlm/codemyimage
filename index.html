The guy who vibe coded the app

I used to send and receive images without MB back then using free messenger (aka lite) but they now give some users the basic plus features to view images using lite. From that I wanted to make this app to convert the image and to text , then send it using lite and then my friend can convert the the text to image or audio

Usage:-
convert the image/audio to text
copy the text and send to your friend using lite messenger
(YOU CAN USE MESSENGER BUT I SAW THAT IT DOES NOT SEND THE WHOLE STRING. THAT'S WHY ITS BETTER TO USE THE LITE MESSENGER)
Your friend has to install the app also to convert the text to Media as well

আগে Messenger Lite / Free Messenger ব্যবহার করে কোনো MB ছাড়াই ছবি পাঠানো যেত। এখন কিছু ইউজার basic plus feature পাচ্ছে, তাই Lite দিয়েও ছবি দেখা যাচ্ছে।
